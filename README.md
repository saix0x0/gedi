# GEDI · Hyderabad Explorer

Explore Hyderabad like a video game. Built for IIITH freshers — scan the QR, pick a quest, go touch grass.

⚔️ Main quests · 🗺️ Side quests · ✨ Specials · XP · levels · badges · Sid's Specials

**Live:** https://saix0x0.github.io/gedi/ · **Build log:** https://saix0x0.github.io/gedi/progress.html

Fully open source, $0 stack: Vite + React + TS + MapLibre GL + CARTO tiles + GitHub Pages.

All places live in `src/data/places.json` — add a place by adding a JSON object, no code changes. See `CONTEXT.md` for architecture.
